To execute the NTautojavadoc script:

	-- Copy autojavadoc.cmd and JavadocRecurse.java to a local directory titled D:\genomics_autojavadoc.
	
	-- Reconfigure the script to use appropriate drive letter and ensure proper paths.
	
	-- Go to the genomics_autojavadoc directory and execute the autojavadoc script from within a DOS shell.

	-- JDK versions are subject to change.
